QueryVista

This prototype website demonstrates a retrieval-augmented generation (RAG) system built for querying user-provided news articles. It combines LangChain for pipeline orchestration, ChromaDB for vector-based semantic search, and GPT-3 via HuggingFace APIs for context-aware answer generation. Users can upload articles, ask natural language questions, and receive accurate, source-grounded responses, significantly improving information accessibility and efficiency when working with large-scale text data.

A Comparative Analysis of Aggregation Methods in Federated Learning on MNIST

Conducted an in-depth analysis of privacy-preserving machine learning methods, focusing on federated learning aggregation techniques (FedAvg, FedProx, FedMedian, q-FedAvg).Designed and trained Convolutional Neural Network (CNN) models on the MNIST dataset to benchmark aggregation strategies under IID and non-IID data distributions. Evaluated convergence patterns, stability, and accuracy trade-offs across methods, identifying FedProx as a consistently robust approach for heterogeneous data environments. Presented findings in AIKP'24 South Africa.
